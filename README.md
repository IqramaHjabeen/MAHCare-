# MAHCare

MAHCare is a healthcare-focused web application built using React (Vite), JavaScript, HTML, and CSS.

## Features
- Search functionality
- Add to Cart functionality
- Responsive design
- Reusable React components
- User-friendly interface

## Technologies Used
- React.js (Vite)
- JavaScript
- HTML5
- CSS3

## Live Demo
https://mahcare.vercel.app/

## Installation
1. Clone the repository
2. Run npm install
3. Run npm run dev
